# What2Eat

What2Eat is a Flutter app that helps you quickly decide what to eat. Save your favorite dishes and swipe through your personal food stack to make a decision.

![App Screenshot](assets/readme/Bild1.png)

---

## ✨ Features

- 📋 Add your own dishes (with image & description)
- 🔄 Swipe-based decision system (similar to Tinder)
- ❤️ Flip cards to view more details
- ♻️ Reset the stack anytime
- 🌙 Light & Dark mode support

---

## 📱 Screens

- **Dishes**: Overview of all saved meals  
- **Swipe**: Interactive decision-making via gestures  
- **Settings**: Theme selection and app info  

---

## 🛠️ Tech Stack

- Flutter  
- Dart  
- SharedPreferences (local storage)  
- Image Picker (gallery/camera)
